# Holomorphic: Interactive Complex Function Visualiser

## Overview
Holomorphic is a visualisation tool for exploring complex-valued functions through contour plots and domain colouring techniques.

It provides an intuitive way to understand how complex functions transform the plane, making abstract mathematical concepts more tangible.

## Features
- Contour plotting of complex functions
- Domain colouring visualisation
- Support for user-defined functions
- Interactive exploration of the complex plane

## Mathematical Concepts
- Complex mappings and transformations
- Magnitude and phase representation
- Conformal mappings
- Singularities and branch cuts

## Visualisation Techniques
- Colour encoding of argument (phase)
- Brightness encoding of magnitude
- Contour overlays for additional structure

## Use Cases
- Understanding behaviour of complex functions
- Visualising roots and poles
- Exploring transformations in complex analysis

## Tech Stack
- Python
- NumPy for numerical computation
- Matplotlib for rendering

## Future Work
- GPU acceleration for real-time rendering
- Animation of function transformations
- Integration with symbolic computation tools
- 3D visualisations

## Motivation
This project was built to bridge the gap between abstract mathematical theory and visual intuition, particularly in complex analysis.
